# Project Manager

Простое веб-приложение для управления проектами и задачами.

## Функции

- Создание проектов
- Добавление задач
- Отслеживание статуса
- Удобный интерфейс

## Технологии

- React
- TypeScript
- CSS

## Установка

```bash
git clone https://github.com/toleukhannov/project-manager.git
cd project-manager
npm install
npm start
